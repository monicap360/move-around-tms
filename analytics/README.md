# Analytics & AI Module

## Overview
This module provides analytics event logging, summary logic, and a dashboard UI for actionable insights. It is ready for AI/ML expansion and integrates with Supabase.

## Key Files
- `analytics.types.ts`: Types for analytics events and summaries
- `analytics.logic.ts`: Core logic for event creation and summary
- `supabase.ts`: Supabase integration for analytics events
- `app/analytics/dashboard.tsx`: Admin dashboard UI for analytics

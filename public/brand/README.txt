MoveAround TMS — Brand Assets

Drop the following logo files in this folder:

  movearound-tms-logo.png
    → Main platform logo. Used in sidebar, login, billing page, and headers.

  ccb-carrier-clearance-bureau-logo.png
    → CCB (Carrier Clearance Bureau) module logo. Used in Module Marketplace CCB card,
      upgrade card, and CCB module page header.

  fast-scan-certified-scanner-logo.png
    → Fast Scan Certified Scanner badge. Used in the Fast Scan page header,
      scanner setup card, and equipment onboarding section.

All three files must be PNG format. Do not rename them — the paths are referenced
in lib/brandAssets.ts and will break if changed.

Acceptable sizes: any resolution. Recommended: at least 300px wide, transparent background.
